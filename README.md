# 勤怠・給与計算自動化システム

Google Apps Script（GAS）と Cloudflare Workers を組み合わせた  
サーバーレス構成の勤怠管理・給与計算自動化システムです。

業務現場で発生する「打刻・承認・集計・給与計算」を  
API連携と自動処理により一気通貫で効率化することを目的としています。


## 解決した業務課題

- 手作業による勤怠集計・給与計算の工数削減
- 入力ミス・計算ミスの防止
- 承認フローの属人化解消
- システム導入コストをかけずにDXを実現


## 3ファイル構成の理由

本システムでは、業務データを以下の3層に分離して管理しています。

- Ops：現場入力・自動取得データ（一次情報）
- Middle：管理者による承認・修正レイヤー
- Admin：承認済データのみを用いた集計・給与計算

この構成により、

- 現場担当者が給与計算ロジックに触れない
- 管理者が原データを直接破壊しない
- 集計・金額データの改ざんリスクを防止

といった **権限分離・事故防止** を実現しています。

単一ファイル構成では起こりやすい
「誤操作によるデータ破損」「権限混在によるトラブル」を防ぐため、
業務システム設計の考え方に沿ってレイヤー構造を採用しています。


## 技術的工夫ポイント

現場入力のフォーマット揺れを想定し、
Opsレイヤーにてデータパース・正規化処理を実装。
