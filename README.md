# 勤怠・給与計算自動化システム

Google Apps Script（GAS）と Cloudflare Workers を組み合わせた  
サーバーレス構成の勤怠管理・給与計算自動化システムです。

業務現場で発生する「打刻・承認・集計・給与計算」を  
API連携と自動処理により一気通貫で効率化することを目的としています。


## 解決した業務課題

- 手作業による勤怠集計・給与計算の工数削減
- 入力ミス・計算ミスの防止
- 承認フローの属人化解消
- システム導入コストをかけずにDXを実現


## システム設計思想（3レイヤー構成）

業務データを以下の3層に分離し、実務に耐える構成としています。

###  Ops Layer（現場入力）
- 勤怠打刻データの受信・保存
- API経由でのリアルタイム登録

### Middle Layer（承認・チェック）
- 管理者による承認・修正

### Admin Layer（給与・管理）
- 承認済データの集約
- 残業・各種手当の自動計算
- 給与計算用データ生成


## 技術的工夫ポイント

現場入力のフォーマット揺れを想定し、
Opsレイヤーにてデータパース・正規化処理を実装。
